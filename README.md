# Smart Queue Management System

## 📌 Overview
A system that manages queues using priority scheduling based on urgency, waiting time, and service type.

## 🚀 Features
- Modular Priority Engine
- Dynamic Priority Updates
- Multiple Queue Management
- Fairness Monitor
- Simulation Mode
- Reporting System
- Admin Configuration

## 🛠️ Technologies
- C++
- Data Structures: Heap, Queue, Hash Table
